Introduction to HTML

HTML ( Hyper Text Markup Language) is used to create document on the World Wide Web. It is simply a collection of certain key words called "tags" that are helpful in writing the document to be displayed using a browser on Internet. HTML web developed by Tim Berners-Lee in 1992.
